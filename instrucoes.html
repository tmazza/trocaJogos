O que o App faz?

ExpChange é um site para conectar pessoas que querem trocas jogos ou consoles de vídeo-game. Após fazer o cadastro é preciso informar quais os itens que você tem disponíveis para troca e quais os itens você está quer receber em troca.

O site irá mostrar o perfil de usuários que satifaçam essas duas condições.

As trocas possuem uma pontuação associada, para orientar se você sair ganhando ou perdendo. Essa pontuação é calculada de acordo com o valor de cada item da troca.

Atividades

  1. Acesse o site: clicando aqui
  2. Faça o cadastro no site
  3. Cadastre pelo menos um item na "lista de desejos"
  4. Cadastre pelo menos um item na "lista de disponíveis para troca"
  5. Encontre uma pessoa para fazer a troca
  6. Faça uma proposta de troca com qualquer usuário
  7. Responda o questionário: clicando aqui
